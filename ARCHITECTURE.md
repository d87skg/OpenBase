
## Hardware Independence

OpenBase sits above the compute layer. It is agnostic to:
Application
|
Agent Framework
|
Runtime
|
====================================
OpenBase (Execution Evidence Layer)
====================================
|
CPU | GPU | NPU | QPU | Storage | Network

text

Core OBS does not change when new hardware emerges.
Extensions (e.g., `obs-quantum`) add domain-specific event types.
