# OpenBase Core Specification v1.0

OpenBase Core 定义了协议的基础层，包含五个不可变的规范文档，构成了 Agent 信任基础设施的核心。

## 核心组件

| 规范 | 文件 | 描述 | 状态 |
| :--- | :--- | :--- | :--- |
| **Evidence** | `evidence.md` | 证据数据结构、签名、哈希链、验证规则 | ✅ 冻结 |
| **Replay** | `replay.md` | 执行重放协议、因果图、保真度等级 | ✅ 冻结 |
| **Registry** | `registry.md` | Runtime 注册、证据提交、证书查询、信任分数 | ✅ 冻结 |
| **Certificate** | `certificate.md` | 证书生命周期、等级、颁发条件、撤销 | ✅ 冻结 |
| **Trust** | `trust.md` | 信任模型、评分维度、时间衰减、冲突检测 | ✅ 冻结 |

## 核心原则

OpenBase Core 遵循以下原则：

1. **不可变性**：一旦冻结，规范不可变更
2. **向后兼容**：所有变更必须保持向后兼容
3. **最小核心**：核心规范保持稳定，不随意增加
4. **可验证性**：所有规范均可通过 Conformance Suite 验证
