# External Contributor Guide

## How to Contribute

### 1. Add an Adapter
- Fork the repo
- Create `adapters/<framework>/emitter.py` following `BaseEmitter`
- Submit PR with test coverage

### 2. Propose an OBEP
- Use RFC_TEMPLATE.md
- Open GitHub Issue with label `OBEP`
- 2-week community discussion period

### 3. Report Bugs
- Use bug_report.md template
- Include minimal reproduction

## First-Time Contributors

Look for issues labeled `good-first-issue`:
- Add event type mapping
- Improve documentation
- Add test coverage

## Review Process

- All PRs require 1 maintainer review
- OBEPs require 2-week discussion + maintainer vote
