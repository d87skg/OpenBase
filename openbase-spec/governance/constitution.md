## 3. Governance Principles

Governance Principles are defined by Canon.

The following principles apply to all governance decisions:

- Mission First
- Protocol First
- Trust First
- Compatibility First
- Knowledge First
- Governance First
- Open Standards First
- Experience First
- Deliver > Discuss
- Preserve Narrative

## 4. Governance Model

All formal changes must pass through the Proposal System.

- RFC: New capabilities or major changes
- ADR: Architecture decisions
- PCP: Protocol specification changes
- Release: Versioned publication

See:

- Proposal Policy
- RFC Process
- ADR Process
- PCP Process
- Release Process

## 6. Amendment

Constitution amendments require:

- Proposal
- Architecture Review
- Protocol Steward Approval

Only then may the Constitution be modified.

Constitution amendments must follow the same Proposal Policy as other governance changes.
