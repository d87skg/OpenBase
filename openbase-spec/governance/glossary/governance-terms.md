# Governance Terms

**Asset-Type:** Governance
**Category:** Glossary
**Status:** Frozen
**Version:** 1.0.0
**Date:** 2026-07-03

---

| Term | Definition |
| :--- | :--- |
| **Proposal** | Any formal change request (RFC, ADR, PCP, etc.) |
| **RFC** | Request for Comments: proposal for new capability or major change |
| **ADR** | Architecture Decision Record: record of an architecture decision |
| **PCP** | Protocol Change Proposal: proposal to modify the Protocol |
| **AR** | Architecture Review: formal review of an architecture design |
| **Review** | Period during which a Proposal is evaluated by the community |
| **Acceptance** | Formal approval of a Proposal by the appropriate authority |
| **Approval** | Final sign-off before Implementation begins |
| **Ratification** | Formal adoption of a Protocol or Policy change |
| **Release** | The versioned publication of an Implementation |
| **Superseded** | Status of a Proposal that has been replaced by a newer one |
| **Archived** | Status of a Proposal that is no longer active |
